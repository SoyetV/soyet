COMPUTER SYSTEM COMPONENTS: MOTHERBOARD

The Motherboard

It is the main printed circuit board in general-purpose computers and other expandable systems. It holds and allows communication between many of the crucial electronic components of a system, such as the Central Processing Unit and memory, and provides connectors for other peripherals.

It is also called the system board because it is the main circuit board of the system unit and it has many electronic components attached to it and others are built into it. It is a computer's central communications backbone connectivity point, through which all components and external peripherals connect. It is the backbone that ties the computer’s components together at one spot and allows them to communicate to each other.

Without it, none of the computer components, such as the CPU, GPU, or hard disk drive, could interact. Total motherboard functionality is necessary for a computer to work well.

NOTE: Computer chips are small pieces of semiconducting material, usually silicon, on which integrated circuits are etched. IC’s contain many microscopic pathways capable of carrying an electric current. They contain components such as resistors, capacitors, and transistors. Manufacturers package chips to be attached to a circuit board, such as the motherboard or adapter cards.

The Motherboard Chipset Layout

Chipset:

It is a set of integrated chips on the motherboard that control communications. After installing, one will need to review a number of elements and parameters: Northbridge cooling, CPU speed comparison and motherboard CPU combos efficiency (or not). These chips control communications between the components of the motherboard and the CPU. The main ones are the Northbridge and Southbridge.

Types of Motherboard

Motherboard Form Factor:

It describes its general shape, the type of case and power supply it can use, and its physical organization (layout of the motherboard). It refers to the physical size and shape (according to outside dimensions) of the board. It also determines additional parameters such as the casing, power supply, mounting holes, and overall physical layout.

NOTE: There are many types of motherboard, and one has to make sure of the correct specification when purchasing it. Generally speaking, the larger the form factor, the more features and support functionality it will have. Micro-ATX and Mini-ITX, however, are perfect for compact builds because their small size still manages to have all the essential features one would need for a quality PC.

The Processor

It is also called the Central Processing Unit (CPU), interprets and carries out the basic instructions that operate a computer. It impacts overall computing power and manages most of the computer’s operations. Most processor chips manufacturers offer multi-core processors (i.e., single chips with two or more separate processor cores). Dual-core: 2 cores Quad-core: 4 cores Each core runs at a slower clock speed than a single-core processor, but still increase the overall performance.

Processor Socket

Motherboards are sub-categorized by the type of processor socket they have. The processor socket (also called a CPU socket) is the connector on the motherboard that houses a CPU and forms the electrical interface and contact with the CPU.

A. PGA Processor Socket

Processor sockets use a Pin Grid Array (PGA) where pins on the underside of the processor connect to holes in the processor socket. Computers based on the Intel x86 architecture include socket processors.

B. LGA Processor Socket

The Land Grid Array (LGA) is a type of surface-mount packaging for integrated circuits (ICs) that is notable for having the pins on the socket (when a socket is used) rather than the integrated circuit.

C. Components of a CPU

The Central Processing Unit (CPU) consists of six main components:

  1. Control Unit (CU)
  2. Arithmetic Logic Unit (ALU)
  3. Registers
  4. Cache
  5. Buses
  6. Clock

Control Unit:

The CU provides several functions:

  • Fetches, decodes, and executes instructions
  • Issues control signals that control the hardware
  • Moves data around the system

Arithmetic Logic Unit:

The ALU performs calculations and makes logical decisions.

  • It performs arithmetic and logical operations (decisions).
  • It acts as a gateway between primary memory and secondary storage.

Registers:

Registers are small amounts of high-speed memory contained within the CPU.

  • They are used by the processor to store small amounts of data needed during processing.
  • Different processors have different numbers of registers for different purposes.

Cache:

Cache is a small amount of high-speed Random Access Memory (RAM) built directly within the processor.

Buses:

A bus is a high-speed internal connection used to send data and instructions between components in the CPU.

Clock:

The clock in the CPU is a timing device that synchronizes all of the processor’s activities.

Motherboard and Processor Overview

CPU System Clock

The CPU system clock is a small quartz crystal circuit used to control the timing of all computer operations. It regulates the speed at which the microprocessor executes instructions, with each tick equating to a clock cycle.

Today, processors are super scalar, meaning they can execute more than one instruction per clock cycle. Clock speed, which is the pace of the system clock, is measured in megahertz (MHz) or gigahertz (GHz).

Current personal computers have clock speeds in the gigahertz range. One megahertz (MHz) equals one million ticks per second, while one gigahertz (GHz) equals one billion ticks per second.

Quartz is a type of crystal used to keep time in watches, computers, and other devices. The crystal vibrates exactly 60 times per minute when electricity is applied.

Two Categories of Processors

32-bit Processor

The 32-bit processor was the primary processor used in all computers until the early 1990s. Intel Pentium processors and early AMD processors were 32-bit, which means that the operating system and software work with data units that are 32 bits wide.

Windows 95, 98, and XP are 32-bit operating systems. A 32-bit processor can only run a 32-bit operating system, and it cannot support a 64-bit OS.

64-bit Processor

The 64-bit processor originated in 1961 with the IBM 7030 Stretch supercomputer, but it wasn't used in home computers until the early 2000s. Microsoft released a 64-bit version of Windows XP for use with 64-bit processors.

64-bit processors can run both 64-bit and 32-bit operating systems, but a 32-bit OS will not use the processor's full capability. A 64-bit processor cannot run 16-bit legacy programs.

Memory

Memory consists of electronic components that store instructions waiting to be executed, data needed by those instructions, and results from processing. It stores operating systems, application programs, and data being processed.

Bytes and Addressable Memory

A byte is the basic storage unit in memory. Instructions and data exist as bytes, and each byte resides temporarily at an address in memory.

Memory Sizes

Manufacturers specify memory size in terms of bytes of available storage.

Types of Memory

  • Volatile Memory - Loses its contents when power is turned off (e.g., RAM).
  • Nonvolatile Memory - Does not lose its contents when power is off (e.g., ROM, Flash Memory).

Random Access Memory (RAM)

RAM is read and written to by the processor and other devices. It is the primary memory used for executing programs and storing operating system files.

Cache Memory

Cache memory stores frequently used instructions and data to speed up processing times. There are L1, L2, and L3 caches with varying speeds and capacities.

Read-Only Memory (ROM)

ROM stores permanent data and instructions, such as firmware, which cannot be easily modified.

Flash Memory

Flash memory is a type of nonvolatile memory that can be electronically erased and rewritten, commonly used in mobile devices.

Complementary Metal-Oxide Semiconductors (CMOS)

CMOS stores BIOS settings, including time, date, and hardware configurations. It is battery-backed to retain information when the computer is powered off.

Memory Access Time

Access time is the amount of time it takes the processor to read data from memory. Faster access times improve the speed of the computer.

BIOS Chip and Battery

The BIOS (Basic Input/Output System) initializes hardware during startup and provides services for operating systems and programs. It is stored on a non-volatile ROM or flash memory chip.

The BIOS performs the POST (Power-On Self Test) to check hardware before loading the operating system. It can also include security features like password protection.

A small CMOS battery powers the BIOS chip to maintain system configuration information like date and time when the computer is off.

Northbridge and Southbridge

Northbridge is the chipset that communicates with the processor, memory, and cache, while Southbridge handles I/O functions like USB and SATA.

IDE and SATA Slots and Ports

IDE and SATA are interfaces for connecting storage devices. SATA is the newer standard and offers faster speeds than IDE.

Expansion Slots

Expansion slots allow additional cards, like video or sound cards, to be added to the motherboard, expanding the computer's capabilities.

Ports and Connectors

Ports and connectors, like USB and FireWire, allow external devices to communicate with the computer. Examples include USB ports, SCSI ports, and HDMI.

Buses

Buses allow devices within the system unit to communicate. The bus width determines how many bits can be transferred at once, and a computer's word size refers to how many bits the processor can handle at one time.
