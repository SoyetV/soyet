SOFTWARE CONCEPTS

Software Concepts

3.0 What is a Software?

Software is a program that enables a computer to perform a specific task, as opposed to the physical components of the system (hardware). This includes application software such as a word processor, which enables a user to perform a task, and system software such as an operating system, which enables other software to run properly, by interfacing with hardware and with other software.

A set of programs, procedures, algorithms and its documentation concerned with the operation of a data processing system.

3.1 Types of Software

The different types of software can be put into categories based on common function, type, or field of use. There are three broad classifications:

  • Application software are the computer programs for performing user tasks such as word processing and web browsers.
    • General Purpose Software
    • Specific Purpose Software
  • System software is used to start and run computer systems and networks. This includes operating systems.
    • System Management Software
    • System Development Software

3.1.2 Application Software

Are programs that are of actual value to the user. From the word “application”, these kinds of software simply “apply” (make use) of the capabilities of the computer (such as high-speed calculations, accuracy given the right algorithm, etc.) to solve anything from simple daily tasks (such as e-mail, word-processing, etc.) to complex special purpose computing (managing the power grid, controlling a nuclear reactor, managing a police database). These can be further classified into two:

  • General-Purpose Application Software - Software that perform common information processing jobs for the end-user, such as; word-processing, presentation management, web-browsing, and many more. They are flexible and may be used for many purposes.
  • Application Specific Software - Software that are designed to accomplish a specific purpose. Examples include ERP, CRM, SCM, and other business applications.

3.1.2.1 General-Purpose Application Software

Software Suites: A collection of functionally related software, often sharing similar GUI, bundled together. Software suites save both money and effort that could have been spent by purchasing individual software. Examples include MS Office, Adobe Creativity Suite, LibreOffice, iWork.

Examples of Office Productivity Suites:

  • Microsoft Office Suite
  • Libre Office Suite
  • iWork Office Suite
  • Adobe Creativity Suite

A. Office Productivity Suites

Are collections of productivity programs intended to be used by knowledge workers. They include:

  • Word Processors - Useful for creating and editing text documents. Includes features such as spell check, grammar check, and table formatting.
  • Electronic Spreadsheets - Used for organizing, calculating, and analyzing data. Allows mathematical calculations and organization into tables.
  • Presentation Software - Used for creating slideshows, often with multimedia components like images and animations.

B. Integrated Development Environment Suites

Programs bundled together for the development of new software. These can include code-generating software, CASE-tools, testing and debugging tools, and more. (NOTE: These fall better under the system software category.)

C. Graphic Processing Suites

Used for image processing, often include tools for image-authoring, motion-processing, or a combination of these applications.

D. Web Browsers

Client software that reads and renders web pages. They have evolved to be more flexible, offering desktop-like applications for various functions such as office productivity, banking transactions, and games.

3.1.2.2 Application Specific Software

Application-specific software packages are customized to meet the needs of particular applications, such as eddy-covariance systems, flood-warning systems, and automated weather stations.

A. Business Application Software

Built to facilitate certain business functions, improving the accuracy, efficiency, and effectiveness of operations. Examples include:

  • Enterprise Resource Planning (ERP)
  • Customer Relationship Management (CRM)
  • Database Management Software (DBMS)
  • Project Management Software (PMS)
  • Business Process Management (BPM)
  • Productivity Software
  • Resource Management Software (RMLS)
  • Time Management Software
  • Educational Software

3.2 System Software

Software that is used to create, run, manage, or support the operation of the computer system itself. Examples include operating systems, hardware drivers, and programming languages.

3.2.1 Types of System Software

A. System Management Software

Software that runs, manages, and/or supports computer hardware, networking, application software execution, and data resources. Examples include operating systems and network management programs.

B. System Development Software

Tools used to create, debug, maintain, or otherwise support other programs and applications. Examples include programming languages.

B.1 Programming Languages

An artificial language designed to communicate instructions to a machine, particularly a computer. Types include:

  • Procedural Programming Language
  • Functional Programming Language
  • Object-oriented Programming Language
  • Scripting Programming Language
  • Logic Programming Language
B.2 Integrated Development Environments

Suites of programs bundled together for the development of new software. This may include code-generating software, CASE-tools, testing and debugging tools, etc.
